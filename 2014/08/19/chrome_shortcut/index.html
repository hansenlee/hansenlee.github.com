常用的chrome快捷键

我用到的chrome快捷键

打开新标签 ctrl + t
切换到指定编号的标签 ctrl + 数字
切换到下一个标签 ctrl + tab
切换到上一个标签 ctrl + shift + tab
关闭当前标签 ctrl + w
当前标签翻页 alt + 左键/右键
打开历史记录 ctrl + h
打开下载记录 ctrl + j
添加到书签 ctrl + d
放大或缩小 ctrl + 鼠标滚轮
向下滚动网页 space
到达网页顶部 Home
到达网页底部 End
将光标定位到地址栏 ctrl + l
