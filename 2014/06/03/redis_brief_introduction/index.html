Redis笔记——基本资料

本文为Redis的读书笔记，主要包括什么是Redis？Redis有什么主要特点？支持哪几种数据结构？

Redis是什么？

Redis（官网），据说是REmote DIctionary Server的缩写，它是一个高性能的非关系型数据库。支持in-memory persistent storage on disk, replication和client-side sharding（sharding：根据ID或Hash Key等唯一标识，对数据进行分片）。性能也颇强大，听说每秒支持好几百gigabyte的数据量和上百万个请求。

Redis与memcached的区别？

memcached也是一个很出名的in-memory非关系型数据库，经常有人拿Redis与memcached进行比较，对两种进行简单比较可以进一步了解Redis的一些特征。

重要特性——持久化存储

Redis server关闭后数据会怎样？
Redis有两种数据持久化存储的方式。
其中一种是point-in-time dump：当满足某条件时（通常是某个时间周期）或执行某个point-in-time dump命令时，将数据写入到硬盘中。
另一种是方式是使用append-only file，执行每个对数据进行改写的操作时，都记录到硬盘文件中。

重要特性——主从复制

Redis支持分布式存储，支持主从复制，每个slave都有一份master的完整独立备份。
